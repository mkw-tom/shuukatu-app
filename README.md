# Entrix　　　　- 就活エントリー管理アプリ　-

[アプリを開く](https://shuukatu-app.vercel.app)

## アプリの機能

### 1. エントリー企業の管理
- エントリーした企業の情報を登録
- 選考フローを可視化
- ボタンひとつで選考ステップをチェック
  
### 2. 選考状況をグラフでかしか
- 円グラフで合格率を確認
- 折れ線グラフで月毎のエントリー状況を確認
  
### 2. AIによる適職診断
- ユーザーの「MBTI」と「能力・志向性」から適した職業を提案
- 能力スコアを算出し、レーダーチャートで確認
